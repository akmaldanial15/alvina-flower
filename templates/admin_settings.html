Telegram Connection

Connect a Telegram bot to receive instant push notifications on your iPhone whenever important events happen.

How to set up Telegram Bot (Step by Step)
  1. Open Telegram on your phone
  2. Search for @BotFather and open it
  3. Send /newbot → name it "Alvina Orders Bot"
  4. BotFather will give you a token — paste it above
  5. Now open your new bot and send /start
  6. To get your Chat ID, visit:
    https://api.telegram.org/bot<TOKEN>/getUpdates
    Look for "chat":{"id": 123456789}
  7. Add recipients below with their Chat ID

WhatsApp Connection (Meta Cloud API)

Connect Meta's official WhatsApp Business Cloud API to receive automated shop notifications directly on your WhatsApp.

How to set up Meta WhatsApp Cloud API (Step by Step)
  1. Go to developers.facebook.com and sign in.
  2. Create a new App (type: Other -> Business) and add the WhatsApp product.
  3. In the left sidebar, navigate to WhatsApp -> API Setup to find your Phone Number ID.
  4. Generate a Permanent Access Token in your Business Manager console.
  5. Go to the WhatsApp Manager -> Message Templates. Create and submit a template (e.g. new_order_alert) with body parameters for order details (e.g. Customer Name, Order ID, Price, Date).
